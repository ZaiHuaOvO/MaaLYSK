# MaaLYSK

基于全新架构**[MaaFramework](https://github.com/MaaXYZ/MaaFramework)** 的恋与深空日常/肝活小助手。图像技术 + 模拟控制，解放双手。我们恋与深空也要有自己的maa！

❗❗❗本质上MaaLYSK是一个图像识别点击器，在电脑的模拟器上读取页面后模拟人手进行操作，不是改后台的科技代码！

图形界面基于 **[MFAAvalonia](https://github.com/SweetSmellFox/MFAAvalonia)**。

## 功能列表

### **🌿日常一条龙**

- ✈️启动/关闭游戏
- ♻切换账号
- 🍰领日常体力
- 📬领好友体力
- 🛍️商城日常领取（每日礼包）
- 🛒商城周常领取（每周礼包，快乐小铺的极空许愿券和辉耀磁石）
- 🌌星间探测
- ⚔️芯核狩猎一次（暂时只支持狩猎一次，按当天开放的芯核区追击）
- ⚔️日常追踪豆佬
- ⚔️日常追踪哈特（默认消耗完体力）
- 🍂日常思念升级一次（会消耗一个绿瓶子，如果排序不当或网卡可能会升级四星/三星卡，介意勿勾选）
- 💞主页面互动
- 📅日常收尾

### **⭐待开发功能（暂未施工，未来可期中）**

- 喵喵牌 娃娃机（周常）
- 陪我一起----运动、睡觉、学习等活动
- 周末时耗空过期体力
- 活动时消耗体力道具以达到300体

### ❗注意事项

- 使用前需在电脑上装好**模拟器**（mumu、蓝叠、雷电等，但是作者只用过mumu，其他模拟器有问题可能需要自行摸索），并在模拟器上面下好恋与深空。
- 页面需为**简体中文**
- 目前`✈️启动/关闭游戏`只支持恋与深空官服（苹果账户模拟器也能使用是互通的），有各位猎人小姐其他服需要支持的话欢迎联系我，看看能不能搞
- 其他项目应该是通用的，但需要在主页面启动
- 目前只支持`win-x86`机型，mac、Linux及win-arm等机型还要等待后续适配（等待一个mac程序大佬加入一起开发调试）

## 使用说明

- 本项目运行环境：

  1、[`Visual C++ 可再发行程序包`](https://aka.ms/vs/17/release/vc_redist.x64.exe)

  2、[`.NET 桌面运行时 10（.NET Desktop Runtime 10.0.0）`](https://dotnet.microsoft.com/en-us/download/dotnet/10.0#:~:text=Binaries-,Windows,-x64)

### **下载步骤:**

1. **本体下载**：release包中的`MaaLYSK-win-x86_64-vx.x.x.zip`

2. **解压**下载的zip文件到某个文件夹

> [!CAUTION]
>
> 不要解压到桌面/C盘，解压整个文件夹到D/E/F盘或者其下的任意文件夹
> 路径不要有中文

1. 打开文件夹，找到两个依赖库安装文件，双击安装

> [!NOTE]
>
> `DependencySetup_依赖库安装.bat`为 `Visual C++`  环境包
>
> `DependencySetup_ 依赖库安装_ win.bat`为 `.net桌面运行时10` 环境包

1. **重新启动**计算机，完成环境安装。

2. 点击文件夹中的**MaaLYSK.exe**文件，开始运行——MaaLYSK启动！

 *任务列表：勾选想做的任务后，MaaLYSK会顺序执行

 *任务设置：中选择配置的选项

 *有任何问题，欢迎提issue或者qq加我私聊反馈！

 ❗勾选任务前请阅读**任务说明**，防止不符合你的预期
