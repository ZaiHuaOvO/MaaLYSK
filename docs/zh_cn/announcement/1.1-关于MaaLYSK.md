## 项目简介

MaaLYSK | **恋与深空**自动化小助手，通过图像识别+模拟控制，MaaLYSK 能自动完成恋与深空中的日常/周常任务、活动任务，帮助你从每日任务中解放双手。

- 软件底层框架基于[MaaFramework](https://github.com/MaaXYZ/MaaFramework)。
- 软件UI界面基于 [**MFAAvalonia**](https://github.com/MaaXYZ/MFAAvalonia)。
- MaaLYSK是免费提供下载使用，禁止以代下载盈利！
- MaaLYSK 属于外置框架并非外挂，本质是精确识图连点器！
  - “加速”、“全局扫荡”、“无限钻石”等功能属于外挂能力，MaaLYSK 不会提供，也坚决抵制此类服务。
  - MaaLYSK仅通过游戏提供的用户界面与游戏程序进行交互，不会读取或修改游戏的**文件或代码**。

## 核心功能

### 🌿 日常一条龙

- ✈️ 恋与深空启动 / ♻ 主页面切换账号
- 🍰 领日常体力（✔月卡免费补领）、📬 好友体力
- 💰 月卡领取（✔鎏金礼）、🛍️ 商城日常领取
- 🌌 星间探测（✔使用加速魔方）
- ⚡️ 体力瓶消耗（✔快过期优先 / ✔全部使用）
- ⚔️ 日常芯核狩猎（✔芯核清理）
- ⚔️ 零点追踪（✔豆佬 / ✔哈特 / ✔黑礼帽 / ✔莱蒙 / ✔呼呼，✔选 1~9 难度，✔选次数）
- 🍂 日常思念升级一次（可指定男主）、💞 主页面互动
- 📅 日常收尾（含鎏金卡补签）

### 🌳 周常任务

- 🛒 商城周常领取（含心砂商店自选购买）、💎 年卡领取
- 📸 拍大头贴 / 拍写真（可自选男主）
- 📓 记录手账（含清空组件选项）
- 💐 密约领取
- 🐱 喵喵牌（✔囤积上限）、🎮 娃娃机（✔秒退 / ✔抓二爪娃娃 / ✔让他抓三爪娃娃）
- 🔗 周常分享 3 次
- 🗓️ 周常收尾

### 🎉 特色功能

- 📲 自动读短信、📞 自动接电话（✔语音 / ✔视频，✔指定男主）、🌐 自动回复朋友圈
- 👥 陪我一起（✔运动 / ✔学习 / ✔睡觉，✔选男主与时长）
- 🏅 喵喵牌抽卡 / 限时抽卡
- 🎁 使用兑换码 、✉️ 邮箱领取
- 📖 自动读剧情

### ⏰ 限时活动

跟随游戏版本更新适配：💫免费福利签到、🛍️限时商城领取、🌟限时活动每日领取、✨读剧情、🌈小游戏等

### ⌛ 待更新内容

混沌深网、灵空行动、家园、爬塔……期待开发者加入！

### 🚧 预设切换

官方提供 5 种预设，可直接载入使用

- 日常任务
- 周常任务
- 特色功能
- 限时活动
- 全部任务列表

可根据需要，在UI页面任务列表处添加/删除/选择任务，为不同账号、不同活动创建不同预设方案，一键长草~
