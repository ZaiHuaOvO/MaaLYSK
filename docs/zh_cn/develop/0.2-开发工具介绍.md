## Maa Pipeline Support

官方使用教程：[专用 VSCode 插件教程 | MAA 文档站](https://docs.maa.plus/zh-cn/develop/vsc-ext-tutorial.html)

### 1. 安装

在VScode插件模块中搜索`Maa Pipeline Support`

<img src="/image/doc_image/0-1/插件下载.png" alt="插件下载" style="width:50%" />

### 2. 连接

打开插件页面，连接模拟器，以进行截图、调试等操作

<img src="/image/doc_image/0-1/插件页面.png" alt="插件页面" style="width:40%" />

编辑好pipeline对应的interface后，想看选项、完整的一个任务能不能正常实现，也在此页面进行任务选择与启动。

### 3. 截图、ROI

打开插件截图小工具：

`Ctrl+Shift+P`打开指令页面，输入maa（或者在maa插件页面点击按钮启动）

<img src="/image/doc_image/0-1/插件打开小工具.png" alt="插件打开小工具" style="width:70%" />

小工具页面：

<img src="/image/doc_image/0-1/插件小工具使用.png" alt="插件小工具使用" style="width:50%" />

### 4. 调试

- 执行某一个节点（`Pipeline`里）

<img src="/image/doc_image/0-1/插件执行.png" alt="插件执行" style="width:50%" />

也可以`Ctrl+Shift+P`打开指令页面，输入maa，在`maa:launch task`中选择对应的节点执行

- 执行整个任务（`interface`配置好任务以及对应选项后）

在VSCode插件工具页面，最下方**`任务`-**选择对应任务，点击**`启动`**执行任务。

::: tip MSE特色功能：

- **插件调试**，在VSCode写好代码，直接对当前节点和当前任务进行调试，并查询匹配度和`box`。
- **滑动**可以直接取到头尾，并直接模拟效果
- 截图小工具，使用对应指令，方便地**截图**、**取ROI**。
- **代码补全**，提供`pipeline`和`interface`的字段提示。

:::

## MaaDebugger

MaaDebugger是MaaFW官方提供的调试工具，采用命令行下载。

官方网站：[MaaXYZ/Maa调试器](https://github.com/MaaXYZ/MaaDebugger)

环境需求：

- Python >= 3.9，<= 3.13
- nicegui >= 2.21，< 3.0

### 1. 下载

```sh
python -m pip install MaaDebugger
```

![maadebugger下载](/image/doc_image/0-1/maadebugger下载.png)

### 2. 使用

```sh
python -m MaaDebugger
```

![maadebugger使用](/image/doc_image/0-1/maadebugger使用.png)

输入指令后，会打开一个网页，在此页面配置，连接模拟器和本地开发文件夹后，即可开始调试

MaaDebugger页面：

![maadebugger调试](/image/doc_image/0-1/maadebugger调试.png)

### 3. 更新

```sh
python -m pip install MaaDebugger MaaFW --upgrade
```

## MFAToolsPlus

项目地址：[MFAToolsPlus： 基于 Avalonia 的 MAAFramework 辅助工具](https://github.com/SweetSmellFox/MFAToolsPlus)

### 1. 下载

从项目地址下载后，解压运行可执行文件，即可使用小工具（和MaaLYSK一样，记得连模拟器）

### 2. 使用

利用此工具，可以很方便地进行取色、截图、取roi、取roi_offset、滑动起终点等操作。

::: tip MPE特色功能：

- **可视化**，方便厘清节点之间的关系。
- **0代码编程**，选择对应字段参数并连线，自动生成pipeline文件。
- 可接入**本地服务**配置，截图、取色小工具齐全，并可视化调试。

:::

## MaaLogAnalyzer

在线网址：[MAA 日志分析器](https://maaloganalyzer.maafw.com/)

项目地址：[MaaLogAnalyzer项目库](https://github.com/Windsland52/MAALogAnalyzer)

### 1. 使用

打开在线网址，上传maa.log即可进行日志分析、文本搜索等操作。

![Maalog网站页面](/image/doc_image/0-1/Maalog分析页面.png)

### 2. BUG排查

有点想咕咕了，先直接看M9A的相关说明吧！[Bug 排查 | M9A 文档站](https://1999.fan/zh_cn/develop/fix.html)

::: tip MLA特色功能：

- MaaFW官方日志分析器，快速准确地定位、分析、解决 bug 。
- **可视化日志分析**，直接定位节点问题。
- 提供api接口，可接入**AI**实现一键分析。
- **多端使用**，网页、VSCode插件、安装包皆可使用，拖入文件即可实现日志分析。

:::
