## 模拟器

> 如果你没有完成下载过游戏，请先完成该部分，前往[恋与深空官网](https://deepspace.papegames.com/home)下载APK
>
> 因为游戏体积很大，可以在下载资源期间继续后续的步骤ฅ•ω•ฅ

- **Windows**

  推荐下载：[MuMu模拟器 12](https://mumu.163.com/)，本项目的开发与测试都是在该模拟器上进行的。

- **Mac**

  M 芯片推荐下载：[蓝叠模拟器 Air 版](https://www.bluestacks.com/mac)

  Intel 芯片及其他模拟器暂无教程提供，如需要可参考 MAA 官方教程：[Mac 模拟器｜MAA 文档站](https://docs.maa.plus/zh-cn/manual/device/macos.html)

  > 本项目暂不支持PlayerCover

> 模拟器下载完成后，请进入设置界面检查以下注意事项
>
> - 分辨率：请设置为`16:9`的比例的分辨率，该比例典型的分辨率有`3840*2160 (4K)`、`2560*1440 (2K)`、`1920*1080 (1080P)`、`1280*720 (720P)`

> - 后台保活运行：关闭

## 下载方式

下载地址：[Releases](https://github.com/Witty36/MaaLYSK/releases)

> 完整 MaaLYSK 软件压缩包命名格式为：MaaLYSK-平台-架构-版本.zip
>
> 例如Windows用户在绝大部分情况下，都需要使用x64架构的 MaaLYSK ，即需要下载 MaaLYSK-win-x64-vx.x.x.zip

- **Windows**

  对于绝大部分用户，请下载  `MaaLYSK-win-x86_64-vx.x.x.zip` ，并解压在非C盘的非中文路径下。推荐解压路径如：`D:\MaaLYSK`

  若确定自己的电脑是 arm 架构，请下载  `MaaLYSK-win-aarch64-vx.x.x.zip`（暂未测试）

- **Mac**

  M 芯片用户请下载  `MaaLYSK-macos-aarch64-vx.x.x.zip`

  Intel 芯片用户请下载 `MaaLYSK-macos-x86_64-vx.x.x.zip`（暂未测试）

- **Linux**

  x64架构请下载 `MaaLYSK-linux-x86_64-v0.1.4.zip`（暂未测试）

  否则请下载`MaaLYSK-macos-aarch64-vx.x.x.zip`（暂未测试）

## 环境安装

> 如果想使用本项目的可视化窗口操作，必须安装 .NET 桌面环境

- **Windows**

  [`Visual C++ 可再发行程序包`](https://aka.ms/vs/17/release/vc_redist.x64.exe)

  [`.NET 桌面运行时 10（.NET Desktop Runtime 10.0.0）`](https://dotnet.microsoft.com/en-us/download/dotnet/10.0#:~:text=Binaries-,Windows,-x64)

  > 可执行项目文件夹目录下的`DependencySetup_ 依赖库安装_ win.bat`文件进行环境安装，等待安装完成后并重启计算机。
  >
  > 若安装失败可点击上方链接跳转并逐个安装。
  >
  > x64 架构，请下载 `.NET 10 Runtime for Windows x64`
  >
  > arm 处理器，请下载 `.NET Runtime 10 for Windows Arm64`

- **MacOS**

  [`.NET 桌面运行时 10（.NET Desktop Runtime 10.0.0）`](https://dotnet.microsoft.com/en-us/download/dotnet/10.0#:~:text=Binaries-,Windows,-x64)

  > 若使用 Intel 处理器，请下载 `.NET 10 Runtime for macOS x64`
  >
  > 若使用 M1, M2 等 arm 处理器，请下载 `.NET Runtime 10 for macOS Arm64`

## 使用方式

- **Windows**

  解压后在文件夹内双击运行 `MaaPiCli.exe`（命令行）或`MaaLYSK.exe`（图形化界面）即可进入软件界面

- **MacOS**

  1. 下载MaaLYSK，解压缩之后，右击文件夹，选择`服务`-`新建位于文件夹位置的终端窗口`

     > 如果没有 Mac 解压软件推荐 `Keka`，App Store 即可下载。

  2. 在终端中依次输入指令

     ```shell
     chmod a+x MaaLYSK
     ./MaaLYSK
     ```

  3. 输入完指令之后会弹出窗口并提示：因为Apple无法检查其是否包含恶意软件

  4. 此时，请前往 `设置` - `隐私与安全性`，下滑至页面底部，找到“已阻止使用的应用”，点击 `仍要打开`/`仍然允许`。请设置窗口保持打开，由于涉及多个文件，需要依次进行允许。

     > 只要隐私与安全性中出现 `仍要打开`/`始终允许`，优先点击，再选择弹窗中的选项 `打开`/`在XX中显示/打开`（不要选择`移到废纸篓`/`取消`）

  5. 再次在终端输入`./MaaLYSK`

  6. 如果没有安装环境或环境安装有问题，就会出现如图这样的下载链接，复制链接到浏览器进行下载安装环境，重新执行`./MaaLYSK`即可
