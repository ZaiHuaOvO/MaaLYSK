## 如何提问/反馈？

不要提**已经在文档站记录，并且拥有完善解决方案的问题**（除非确实解决不了你的）

不要提**太过基础的问题**，例如**软件在哪里下载**

❗❗请确保你的问题是**稳定触发**而非**偶发事件**（因网络卡顿、模拟器卡死、未成功进行连接等原因造成的个例）后再进行提问

📌如果有报错信息可以尝试复制给deepseek，AI有时候比人好使多了。（甚至你发给项目组，项目组也转头去问AI）

提问时请尽可能详细描述你所遇到的问题、你的环境、你的行为

提问步骤：

1. **遇见问题，查询文档，确认文档无法解决自己的问题**
2. **尝试性网络搜索、询问Deepseek并解决（可选）**
3. **无法解决，提出问题**

## 下载/安装问题

### MaaLYSK下载太慢/下载失败

因为Github在国内网络访问速度很慢，所以会出现下载失败/无法下载等问题。

不使用Github？推荐以下两种方式：

- MaaLYSK交流群：1065676561，群文件通常会更新最新版本的MaaLYSK，但每次更新都需要重新下载。

- Mirror酱（付费）：[下载链接](https://mirrorchyan.com/zh/projects?rid=MaaLYSK)，购买CDK后可以使用高速网络下载MaaLYSK，并且每次软件都会自动高速更新。

  如果你不知道如何使用，请前往：[Mirror酱使用说明](/zh_cn/manual/2.1.md)

### 系统问题

- `MaaLYSK`不支持 32 位操作系统，不支持 Windows 7 / 8 / 8.1。
- 安装依赖系统组件（CBS/TrustedInstaller/WinSxS），若组件损坏需重装系统，请避免使用精简版或老旧系统。

### 什么时候要更新

- 想要使用新出功能时；
- 当前版本运行出现失误/故障等情况时；

## 启动问题

### MaaLYSK打不开/直接报错

大概率都是运行库安装问题，具体参照[快速开始--环境安装](https://maalysk.top/zh_cn/manual/1.1.html#环境安装)和[图文教程--环境准备](https://maalysk.top/zh_cn/manual/1.3.html#环境准备)进行安装。

### 提示“Windows已保护你的电脑”、“xx阻止了未知应用启动”

在页面寻找`更多`-`继续运行`/`保留`等类似字样，继续运行

<img src="/image/tutorial_image/Q&A/阻止运行弹窗.png" alt="阻止运行弹窗" width="70%" />

### 提示”同一路径下只能启动一个实例！“

- **想要多开：**

  v0.3.0以前版本：复制`整个MaaLYSK文件夹`，粘贴到空白处使用

  V0.3.0及以后版本：点击MaaLYSK页面上方`+`号，`添加实例`后。选择对应模拟器

- **不双开：**打开MaaLYSK时点多了，直接关掉

- **没点多，但是提示这个，也没找到其他maa（记得图标栏也找一下）：**任务管理器关闭全部MFAAvalonia.exe后，重新打开MaaLYSK，或重启电脑

### 报错：此应用无法在你的电脑上运行

误下了ARM64版本（不兼容 Windows x64），请重新下载文件名含 `MaaLYSK-*-win-x64.zip` 的压缩包。

### 报错：无法加载dll

- **操作错误**

  可能是由于没有解压/直接把`MaaLYSK.exe`拖到了桌面导致的。

  确保在已经将MaaLYSK安装包进行解压，并在解压后的文件夹内运行`MaaLYSK.exe`。

  不要把`MaaLYSK.exe`直接拖到桌面，而是选中后右击-`发送到`-`桌面快捷方式`以便于在桌面点击使用。

<img src="/image/tutorial_image/Q&A/桌面快捷方式创建.png" alt="桌面快捷方式创建" width="70%" />

- **运行库安装问题**

  可能是运行库安装错误导致，具体参照[快速开始--环境安装](https://maalysk.top/zh_cn/manual/1.1.html#环境安装)和[图文教程--环境准备](https://maalysk.top/zh_cn/manual/1.3.html#环境准备)进行安装。

## 运行问题

### 运行任务后无反应或持续点击模拟器某处

确保MaaLYSK为最新版本。

- **模拟器设置问题**

  - 分辨率设置错误

    确保分辨率是`900:1600`且模拟器是竖屏（`手机`）而非横屏（`平板`）。具体设置方法参照[图文教程--模拟器设置

  - 关闭模拟器后台保活

    例如 MuMu 模拟器：右上角`菜单` → `设置中心` → 其他 → 取消勾选`后台挂机时保活运行`。

- **MuMu模拟器本身bug**

  MuMu模拟器本身的竖屏模式有一些bug，导致在使用时会出现点击错位的问题。

  尝试在MaaLYSK`设置`-`连接设置`-`触控方式`中切换成其他触控方式，看看情况有没有好转

![触控方式选择](/image/tutorial_image/Q&A/触控方式选择.png)

- **多开冲突**

  确认所需连接的模拟器端口与MaaLYSK显示端口一致。具体参照[图文教程--连接设置](https://maalysk.top/zh_cn/manual/1.3.html#连接设置)进行连接。

- **没招了**

  新建一个竖屏模拟器，记得MaaLYSK连接刷新一下连对应模拟器
  
  重新安装MaaYuan和模拟器。

### 截图画面撕裂，MaaLYSK不工作或点击错位

- 在MaaLYSK里`设置`-`运行设置`-`截图测试`，如果截图测试的画面呈撕裂状时，就会不工作或点错位置
- 原因：MaaLYSK启动时获取到的分辨率与后续不一致。例如：模拟器一开始是横屏启动的，打开深空后变成了竖屏。
- 解决方案：
  确保分辨率是`900:1600`且模拟器是竖屏（`手机`）而非横屏（`平板`）。具体设置方法参照[图文教程--模拟器设置。
  重启模拟器。

### 任务0S完成

- 大概率是”伪连接“了模拟器----看似是连接良好实际上连接已经断开了。点击模拟器连接板块的`刷新`按钮，并选择对应模拟器，或进行手动连接。具体参照[图文教程--连接设置](https://maalysk.top/zh_cn/manual/1.3.html#连接设置)进行尝试。
- ADB占用冲突：参照[常见问题 | 连接问题](https://maalysk.top/zh_cn/manual/2.2.html#连接问题)尝试解决。
- 多开冲突：电脑性能落后或多开造成冲突。
- 进程卡住：关闭MaaLYSK，不用MaaLYSK打开模拟器，尝试手动打开模拟器后再打开MaaLYSK，正常连接后尝试问题是否解决。
- 仍然不行：重启电脑-重启模拟器-重新下载MaaLYSK最新版

### 截图用时过长/触控反应时间过长

- 截图耗时≤1000ms属正常。
- 启用`GPU加速推理`（MaaLYSK`设置` - `高级设置`），核显/老旧独显（如GTX 750Ti以下）不建议开启；遇bug则关闭。
- **检查分辨率**：分辨率应为`手机`-`900*1600`-`DPI320`。

### 设置页面无法编辑

任务运行时设置页面按钮会呈现灰色，需停止任务进行编辑

### MaaLYSK占用过大

- 在MaaLYSK`设置`-`运行设置`，将几个`截图（用于调试）`的选项关闭。

- 定期删除`debug`文件夹（注意：在**关闭MaaLYSK后**再删除）

- ![触控方式选择](/image/tutorial_image/Q&A/debug文件夹位置.png)

### 雷电模拟器

- **关闭自动旋转屏幕**：设置 → 其他设置 → 关闭。
- **调整分辨率**：设置 → 性能设置 → 推荐「`900:1600`-竖屏（`手机`）」

### **连接正常，但是操作卡顿、异常、频繁出错或**显示任务已完成

- 异形屏UI适配设为0，并保证分辨率正确。
- 主页 → 启动游戏 → 齿轮 → 选择正确游戏客户端版本。
- Adb Input触控模式慢属正常现象。
- 尝试重启电脑、模拟器、MaaLYSK，并保持MaaLYSK为最新版本。

- 关闭模拟器后台保活；
- 模拟器旧版本可能连接不上，请更新模拟器

### 更新过后连不上模拟器

- **提示资源加载失败** → 删除现有`MaaLYSK文件夹`，重新下载/解压

- **能搜到模拟器** → 点`自定义`→`保存`

- **搜不到模拟器** → 重启（按顺序尝试）：MaaLYSK→ 模拟器 → 电脑

- **ADB异常** → 强制关闭ADB工具后重新搜索

### 手动连接模拟器

- **填写信息**：`MaaLYSK`主界面 - 右上角连接框内 - 点击自定义按钮，填 adb 地址及路径；

<img src="/image/tutorial_image/Q&A/模拟器连接自定义.png" alt="模拟器连接自定义" width="90%" />

- **导入 ADB路径**：点击 Adb 路径后的“导入”，在模拟器的安装路径中找到 adb 的 exe 文件（如 adb.exe），点击打开；

<img src="/image/tutorial_image/Q&A/ADB自定义.png" alt="ADB自定义" width="90%" />

> [!WARNING]
>
> 若用自定义连接，勿点“刷新”或“自动检测”（避免路径被重置）。

- **填写模拟器ADB地址**：本机模拟器连接地址为 127.0.0.1:<端口号> 或 emulator-<四位数字>。

可以尝试将额外参数删掉，填写好ADB路径和ADB地址并保存后，让MaaLYSK自己寻找并连接模拟器。

::: tip **常见模拟器ADB端口地址**

端口号查询方法`自行上网搜索`或`询问模拟器客服`

常见模拟器端口号如下：

- BlueStacks 5：`127.0.0.1:5555/5556/5565/5575/5585/5595/5554`

- 雷电模拟器 9：`emulator-5554/5556/5558/5560`, `127.0.0.1:5555/5557/5559/5561`
- 夜神模拟器：`127.0.0.1:62001/59865`
- 逍遥模拟器：`127.0.0.1:21503`

::: details 其他adb连接问题的可能解决方法

**替换Adb**（解决连接/报错问题）：

- 下载[platform tools](https://wwsq.lanzoue.com/i0Yhg2gk5czg)并解压到**全英文路径**；
- 打开`MaaYuan`设置 → 连接 → 点击Adb路径图标 → 选择解压文件夹内的**adb.exe**（*不可单独使用，需保留完整文件*）；

**防火墙问题**：临时关闭防火墙，并将 **adb.exe** 加入白名单。

**路径问题**：确保`MaaLYSK`和`platform tools`所在目录**无中文/特殊字符**（改用全英文路径）

### 多开连接问题（实时视图和模拟器页面对不上）

MaaLYSK不会自己寻找连第二个模拟器，需要你确认多开模拟器的端口号并连接对应模拟器。

想要实现MaaLYSK多开：复制整个MaaLYSK文件夹，把两个文件夹中的`可执行文件`都新建快捷方式到桌面。

自动寻找多开模拟器：

1. 点击MaaLYSK模拟器连接页面`刷新`按钮，寻找模拟器。
2. 点击`下拉`按钮，找到对应模拟器`端口号`并选择。

手动填写:参照[常见问题 |连接问题-手动连接模拟器](<https://maalysk.top/zh_cn/manual/2.2.html#手动连接模拟器>)

内置更新源分为Github和Mirror，两者都有一定使用条件。
