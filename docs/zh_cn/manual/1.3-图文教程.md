## 下载MaaLYSK

> [!IMPORTANT]
>
> 如果你没有完成下载过游戏本体，请先前往[恋与深空官网](https://deepspace.papegames.com/home)下载APK
>
> 选择`Android下载`并放在你能找到的地方
>
> 因为游戏体积很大，你可以在下载资源期间阅读后续的步骤ฅ•ω•ฅ

下载MaaLYSK：[Releases](https://github.com/Witty36/MaaLYSK/releases)

> 完整 MaaLYSK 软件压缩包命名格式为：MaaLYSK-平台-架构-版本.zip
>
> 例如Windows用户在绝大部分情况下，都需要使用x64架构的 MaaLYSK ，即需要下载 MaaLYSK-win-x64-vx.x.x.zip

- **Windows**

  对于绝大部分用户，请下载  `MaaLYSK-win-x86_64-vx.x.x.zip` ，并解压在非C盘的非中文路径下。推荐解压路径如：`D:\MaaLYSK`

  若确定自己的电脑是 arm 架构，请下载  `MaaLYSK-win-aarch64-vx.x.x.zip`（暂未测试）

- **Mac**

  M 芯片用户请下载  `MaaLYSK-macos-aarch64-vx.x.x.zip`

  Intel 芯片用户请下载 `MaaLYSK-macos-x86_64-vx.x.x.zip`（暂未测试）

- **Linux**

  x64架构请下载 `MaaLYSK-linux-x86_64-v0.1.4.zip`（暂未测试）

  否则请下载`MaaLYSK-macos-aarch64-vx.x.x.zip`（暂未测试）

## Windows

推荐使用[MuMu模拟器12](https://mumu.163.com/)，后续步骤均会以该模拟器进行讲解。

### 模拟器设置

1. 新建一个设备（随便设置）后启动模拟器，选择右上角的三个点，选择`设备设置`

2. 在`性能`-`性能设置`中，处理器设置为`2核及以上`，运行内存设置为`2GB及以上`

   ![模拟器性能设置](/image/tutorial_image/win/模拟器设置0.png)

   > [!NOTE]
   >
   > 性能通常推荐设置得越高越好~~（因为恋与深空真的很卡）~~
   >
   > 如果模拟器出现明显的卡顿、卡死现象，请及时提升性能，可能会影响到Maa的识别和操作。

3. 在`显示`-`分辨率设置`中，选择手机，并选择任意一个分辨率，推荐`1080*1920`

   ![模拟器分辨率设置](/image/tutorial_image/win/模拟器设置1.png)

4. 在`其他`-`应用保活`中，确认`后台挂机时保活运行`处于**未勾选**（关闭）状态

   ![模拟器应用保活](/image/tutorial_image/win/模拟器设置2.png)

5. 点击设置界面右上角的`X`后，选择`立即重启`

   ![模拟器重启](/image/tutorial_image/win/模拟器设置3.png)

### 下载游戏

下载地址：[恋与深空下载](https://deepspace.papegames.com/home)

1. 选择 Android 下载，并放在你知道的地方。

2. 选择右上角的三个点，选择`更多工具`-`APK安装`，找到APK安装包并确认

   ![安装游戏](/image/tutorial_image/win/下载游戏0.png)

3. 安装完成后打开，下载资源。

### 环境准备

可执行项目目录下的`DependencySetup_ 依赖库安装_ win.bat文件`进行环境安装，等待安装完成后并重启计算机。

若安装失败可点击下方链接跳转并逐个安装。

- [`Visual C++ 可再发行程序包`](https://aka.ms/vs/17/release/vc_redist.x64.exe)
- [`.NET 桌面运行时 10（.NET Desktop Runtime 10.0.0）`](https://dotnet.microsoft.com/en-us/download/dotnet/10.0#:~:text=Binaries-,Windows,-x64)

x64 架构，请下载`.NET 10 Runtime for Windows x64`（如果你不知道选择哪个，就下载这个）

arm 处理器，请下载`.NET Runtime 10 for Windows Arm64`

   ![环境准备](/image/tutorial_image/win/环境准备0.png)

> [!NOTE]
>
> 请确保你所下载的是`.NET Runtime x.x.x`
>
> 不是`SDK`、不是`ASP.NET Core Runtime`、也不是`.NET Desktop Runtime`

### 连接设置

当你第一次启动MaaLYSK时，它会自动识别并查找你的模拟器地址

![连接设置](/image/tutorial_image/win/连接设置.png)

如果出现了意外（连接失败、找不到模拟器），可以尝试点击刷新按钮（右上角三个按钮中间的按钮）

也可以选择手动查找（右上角三个按钮左边的按钮），手动选择MuMu模拟器的应用程序本体

通常默认的安装地址：`C:/Program Files/Netease/MuMuPlayer-12.0`

## MacOS

M 芯片推荐[蓝叠模拟器 Air 版](https://www.bluestacks.com/mac)，后续会只按照该模拟器进行讲解。

> 本项目不支持 PlayerCover，而MuMu 模拟器要钱~~（把 mac 用户当韭菜呢）~~

Intel 芯片及其他模拟器暂无教程提供，如需要可参考 MAA 官方教程：[Mac 模拟器｜MAA 文档站](https://docs.maa.plus/zh-cn/manual/device/macos.html)

> Intel 芯片推荐使用 Mac 自带的多系统安装 Windows ，并使用 Windows 版

### 模拟器设置

1. 选择右上角的三根横线，选择`设置`

   ![模拟器横线](/image/tutorial_image/mac/模拟器横线.png)

2. 设置中文

   ![模拟器设置中文](/image/tutorial_image/mac/模拟器设置中文.png)

3. 设置 ADB

   ![模拟器设置ADB](/image/tutorial_image/mac/模拟器设置ADB.png)

4. 在性能中，设置为`高性能`（可选）

5. 设置分辨率后重启

   ![模拟器设置分辨率](/image/tutorial_image/mac/模拟器设置分辨率.png)

### 下载游戏

下载地址：[恋与深空下载](https://deepspace.papegames.com/home)

1. 选择 Android 下载，并放在你知道的地方。

2. 下载完成后在模拟器右侧选择 APK 按钮，找到APK安装包并确认

   ![模拟器APK安装](/image/tutorial_image/mac/模拟器APK安装.png)

3. 安装完成后打开，下载资源。

### 环境准备

1. **运行库安装**：[下载链接](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)，下载后直接运行安装

   > 若使用 Intel 处理器，请下载 `.NET 10 for macOS x64`
   >
   > 若使用 M1, M2 等 arm 处理器，请下载 `.NET 10 for macOS Arm64`

   ![环境安装0](/image/tutorial_image/mac/环境安装0.png)

2. 下载MaaLYSK，解压缩之后，右击文件夹，选择`服务`-`新建位于文件夹位置的终端窗口`

   > 如果没有 Mac 解压软件推荐 `Keka`，App Store 即可下载。

   ![环境安装1](/image/tutorial_image/mac/环境安装1.png)

3. 在终端中依次输入指令

   ```shell
   chmod a+x MaaLYSK
   ./MaaLYSK
   ```

4. 输入完指令之后会弹出窗口并提示：因为Apple无法检查其是否包含恶意软件……

   ![环境安装2](/image/tutorial_image/mac/环境安装2.png)

5. 此时，请前往 `设置` - `隐私与安全性`，下滑至页面底部，找到“已阻止使用的应用”，点击 `仍要打开`/`仍然允许`。请设置窗口保持打开，由于涉及多个文件，需要依次进行允许。

   ![环境安装3](/image/tutorial_image/mac/环境安装3.png)

   > 只要隐私与安全性中出现 `仍要打开`/`始终允许`，优先点击，再选择弹窗中的选项 `打开`/`在XX中显示/打开`（不要选择`移到废纸篓`/`取消`）

6. 再次在终端输入`./MaaLYSK`

7. 如果没有安装环境或环境安装有问题，就会出现如图这样的下载链接，复制链接到浏览器进行下载安装环境，重新执行`./MaaLYSK`即可

   ![环境安装4](/image/tutorial_image/mac/环境安装4.png)

### 连接设置

1. 设置快捷方式，在项目文件夹中找到`MaaLYSK`，右键选择`制作替身`，生成的新文件就是本项目的快捷方式了，双击即可打开`MaaLYSK`（放在程序坞、快捷栏均可）

   <!-- ![软件设置0](/image/tutorial_image/mac/软件设置0.png) -->

2. 如果你已经安装了模拟器，MaaLYSK 通常可以顺利识别，如图所示（5554是蓝叠模拟器的默认端口）。

   > 如果识别失败，可以点击刷新按钮重新扫描，或手动选择应用程序中的`BlueStacks`

   ![软件设置1](/image/tutorial_image/mac/软件设置1.png)
