# <i class="ri-recycle-line"></i> 更新说明

## 更新说明

目前更新方法一共三种，Mirror酱更新，Github更新，软件包整包更新。

其中，Mirror酱更新、Github更新为内置更新，在软件内进行相关设置后即可自动更新；Mirror酱为第三方更新付费平台，需购买CDK之后使用。

> [!NOTE]
>
> **【最省事】** Mirror酱-购买CDK实现内置更新。点击此处跳转[Mirror酱使用说明](https://maalysk.top/zh_cn/manual/2.1.html)<br>【最多选择】删除旧的安装包，网盘重新下载 。<br>【科学上网】GitHub内置更新，需自行购买加速服务或搭建VPN。
>
> **内置更新：**
>
> 在MaaLYSK`设置`-`更新设置`中，选择下载源并进行相关配置后，直接自动更新或点击按钮手动更新。
>
> - 自动更新：软件设置-更新设置中，打开`自动检测更新`和`自动更新资源`按钮，即可实现自动更新。
>
> - 手动更新：软件设置-更新设置中，关闭`自动更新资源`按钮，点击`检查资源`和`资源更新`右侧的刷新按钮即可实现手动更新。
>
> ![更新设置](/image/tutorial_image/update/更新设置.png)
>
> **外置更新：**
>
> 手动下载最新的安装包，删除旧的MaaLYSK包体。
>
> 安装包可以在MaaLYSK交流QQ群中寻找，群号：1065676561

### Github更新

Github更新为软件内置更新，但是由于需要连接到Github本站，速度较慢，需要借助某些**科学手段**，否则很容易更新失败。

无法Github更新的，请选择外置更新、购买mirror酱、研究科学上网。

更新方法：

- 打开MaaLYSK，在设置-更新设置中，将`下载源`选为`Github`。
- 点击`资源更新`按钮等待更新完成。

其他Github更新问题参照[常见问题 - 更新问题](https://maalysk.top/zh_cn/manual/2.2.html#github更新提示rate-limit-exceeded)。

### Mirror酱更新

Mirror酱更新为第三方平台更新，速度很快，但需要付费使用。

更新方法：

- 在MaaLYSK的`设置`-`更新设置`中设置下载源为`Mirror酱`，并填写你的mirror酱CDK。

- 打开`自动检测更新`和`自动更新资源`。

- 可点击`检查资源`或`资源更新`右侧的`刷新`按钮来手动更新资源。

> Mirror酱购买：点击CDK旁边的`关于Mirror酱`，跳转到[Mirror酱购买链接](https://mirrorchyan.com/zh/get-start?source=MaaLYSK_docUpdate)
>
> 更多关于Mirror酱的问题，参照[Mirror酱使用说明](https://maalysk.top/zh_cn/manual/2.1.html)

### 软件包更新

手动下载最新的安装包并解压到**另外的空文件夹**，然后把旧的MaaLYSK**整包**删除。

如果自己选了**配置**（任务列表齿轮、`设置`中相关配置），在`你的MaaLYSK文件夹\config`下面有你自己的配置，复制到新文件夹使用即可。

> [!WARNING]
>
> 如果旧配置文件迁移后出现问题，请放弃迁移，重新进行配置选择。

## MaaLYSK版本说明

### 版本查看方法

1、`v0.3.8`版本及之后：UI页面标题后`v...`内容即为版本号。

2、MaaLYSK`设置`-`更新设置`，查看`资源版本`

</div>

:::

### 命名方式

正式版：`vx.y.z`，无任何后缀

公测版：`vx.y.z-batah`，含有`beta`后缀

~~内测版:`vx.y.z-alphah`，含有`alpha`后缀~~

::: tip 📌如何判断是否为最新版本

MaaLYSK版本号采用[语义化版本](https://semver.org/lang/zh-CN/)规范，由`vX.Y.Z`+`先行版本号`构成。

<div align="center">
  <img src="/image/tutorial_image/update/版本说明.png" alt="版本说明" width="70%" />
</div>

判断方法：

1. 从左至右**依次**判断`X`、`Y`、`Z`、`H`的**数值**大小，**第一个差异值**决定优先层级。
2. 当主版本号、次版本号及修订号**都相同**时，改以优先层级比较低的**先行版本号**决定。

> 例：v0.1.9 < v0.2.1 < v0.3.6 < v0.3.9-beta1 < v0.3.9-beta3 < v0.3.9

:::

### 版本区别

【✅**正式版**】只接收经过多日公测和修复后基本能够保障稳定运行的新功能。

【🌐**公测版**】第一时间接收到最新功能和 bug 修复，但也可能品尝到最新功能带来的最新 bug。

~~【🚧**内测版**】 MaaLYSK 不直接推送内测内容，**请勿选择此选项**。~~

> [!IMPORTANT]
>
> ❓️什么时候会出公测版？
>
> 更新UI页面、打包方式，或常驻功能优化了结构代码时，由于可能会出现未知问题，所以进行公测版发布。
>
> 公测版一般为最新版本，如果想使用限时活动，则需更新公测版。不打算用最新功能、只使用常驻功能的话，可以使用正式版。
>
> 正式版遇到bug请更新到最新公测版，公测版仍有问题再参照[常见问题 | 提问方式](https://maalysk.top/zh_cn/manual/2.2.html#如何提问-反馈)进行反馈。

### 选择版本方法

#### 内置更新

在`设置`-`更新设置`-`资源更新渠道`中，选择对应想要更新的版本即可。

#### 外置更新

下载对应安装包。
